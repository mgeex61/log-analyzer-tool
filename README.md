# 🧠 Log Analyzer Tool

Ein einfaches Web-Tool zur Analyse von Log-Dateien.  
Es erkennt automatisch Fehler, Warnungen und Informationsmeldungen und stellt diese übersichtlich dar.

---
🔍 Live Demo: https://mgeex61.github.io/log-analyzer-tool/

## 🚀 Funktionen

- Analyse von Log-Daten (Text oder Datei)
- Erkennung von:
  - ❌ Fehlern (ERROR / FEHLER)
  - ⚠️ Warnungen (WARNING / WARNUNG)
  - ℹ️ Informationen (INFO)
- Anzeige von Gesamtanzahl und Kategorien
- Upload von `.txt` und `.log` Dateien
- Übersichtliche Darstellung der Ergebnisse

---

## 💻 Technologien

- HTML
- CSS
- JavaScript

---

## 📂 Nutzung

1. Projekt herunterladen oder klonen
2. `index.html` im Browser öffnen
3. Log-Daten einfügen oder Datei hochladen
4. "Analysieren" klicken

---

## 📄 Beispiel Log-Datei
2026-04-25 ERROR: Verbindung fehlgeschlagen
2026-04-25 INFO: Benutzer angemeldet
2026-04-25 WARNING: Antwortzeit hoch
2026-04-25 ERROR: Timeout
